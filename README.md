# AI Directory - Deployment Ready

A monetized AI tools directory website with Google AdSense integration.

## 🚀 Deploy to Vercel (Free)

### Option 1: Drag & Drop
1. Go to [vercel.com](https://vercel.com) and sign in
2. Click "Add New" → "Project"
3. Drag & drop this folder
4. Click "Deploy"
5. Get your free URL!

### Option 2: GitHub
1. Push this folder to GitHub
2. Import the repository in Vercel
3. Click "Deploy"
